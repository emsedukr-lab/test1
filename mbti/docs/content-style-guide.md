# 카드 콘텐츠 스타일 가이드 (필수 준수)

## 문체

- 합니다체 평서문, 주어 생략형. 독자를 단정하지 않고 가능성으로 말한다.
- 허용 종결형 예: ~할 수 있습니다 / ~가능성이 있습니다 / ~시기입니다 / ~로 보입니다 / ~흐름입니다 / ~읽힙니다 / ~만합니다
- 한 카드 안에서 같은 종결어미를 3회 초과 사용 금지. 어미를 다양하게.
- 2인칭 명령형 금지: '하세요' 대신 '해 보세요'.

## 금지 표현 (검증 스크립트가 잡음)

반드시 / 무조건 / 절대로 / 틀림없 / 확실합니다 / 100% / 헤어집니다 / 헤어지게 됩니다 / 이별하게 됩니다 / 성공합니다 / 실패합니다 / 망합니다 / 불행해집니다 / 저주 / 벌을 받 / 당신은 원래 / 사랑하지 않습니다

또한: 미래 확정, 공포 조장, 의료·법률·투자 판단 제시 금지. 부정적 카드(죽음, 타워, 소드 10 등)도 공포가 아닌 변화·직면·회복의 관점으로 쓴다.

## 필드 규칙 (TarotCard)

- keywords: 3개 이상, 명사(구)만 (예: '새로운 시작', '자유로움')
- summary: 한 문장, 20~90자
- lightMeaning / shadowMeaning / relationshipMeaning / careerMeaning / moneyMeaning / selfGrowthMeaning / decisionMeaning: 각 1~2문장 완결문, 30~160자. careerMeaning에는 '직장'이라는 단어 금지(학업 분야에서 재사용되므로 '일과 과제', '맡은 일' 등 범용 표현 사용)
- strengths: 2개 이상, 명사형 종결 (예: '가볍게 시작하는 힘')
- cautions: 2개 이상, 명사형 종결 (예: '준비 없이 뛰어들기 쉬움')
- reflectionQuestions: 3개, 반드시 '~까요?' '~나요?' '~가요?' 로 끝나는 의문문
- suggestedActions: 3개, 반드시 '~해 보세요' 또는 '~볼까요' 로 끝남. 24시간~7일 안에 실행 가능하고 사용자가 통제 가능한 구체적 행동(구체적 동사, 작은 단위). 예: '미뤄둔 새로운 시도 하나를 오늘 10분만 가볍게 시작해 보세요.'
- image: 정확히 '/cards/<카드id>.jpg'
- imageAlt: 라이더-웨이트 그림 장면을 간결히 묘사 + '— 라이더-웨이트 <카드명> 카드' 형식

## 다양성 규칙

- 같은 파일 내 카드끼리 suggestedActions / reflectionQuestions 문장이 겹치지 않게 각 카드 맥락에 밀착시킨다.
- '점검해 보세요' 같은 범용 행동을 반복하지 말고 카드 상징에서 나온 구체 행동을 쓴다.
